---

Ты помогаешь нетехническому пользователю развернуть это FastAPI-приложение локально.
Веди его пошагово, простым языком, по-русски. После каждого шага коротко говори, что произошло.

1. Проверь версию Python (нужен 3.11+). Если нет — объясни, как поставить под его ОС, и остановись.
2. Создай виртуальное окружение и установи зависимости из `requirements.txt`.
3. Создай `.env` из `.env.example`.
4. По очереди спроси у пользователя `OZON_CLIENT_ID` и `OZON_API_KEY`:
   - объясни простыми словами, что это и где взять (кабинет продавца Ozon → Настройки → Seller API → создать ключ, роль «Admin read only»);
   - попроси вставить значение в чат;
   - впиши его в `.env` сам. НЕ выводи значения ключей обратно в чат.
5. Запусти приложение: `uvicorn dashboard.app:app --port 4000`. Дождись успешного старта.
6. Дай пользователю ссылку http://localhost:4000 и объясни, что открыть её в браузере и нажать «Обновить данные».

Если у пользователя пока нет ключей Ozon — предложи запустить `python seed_demo.py` (демо-данные) и затем приложение.

Если на любом шаге ошибка — объясни причину простыми словами и предложи исправление, не сдавайся молча.
