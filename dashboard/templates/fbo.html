#
Порядковый номер в текущей сортировке.
Артикул / Название
Артикул продавца (offer_id) и полное название товара из каталога Ozon. Кликните для сортировки.
ABC
Двухфакторный ABC-анализ. Первая буква — по выручке за 30д (р). Вторая — по количеству за выбранный период (кнопки 10/28/30д вверху). A = топ 80% продаж, B = следующие 15%, C = остальные 5%. Используйте фильтры ABC ₽ и ABC шт чтобы выбрать нужные категории.
Факт FBO
Текущий остаток на складах Ozon (FBO) суммарно по всем кластерам. Кликните для сортировки.
В пути
Товары, которые уже отгружены и едут на склады Ozon, но ещё не приняты. Учитываются в расчёте рекомендации (факт + в_пути = доступный сток). Кликните для сортировки.
Дней
На сколько дней хватит текущего стока при среднем темпе продаж. Красный < 7 дн = дефицит, жёлтый < 14 дн = риск. Кликните для сортировки.
Спрос/д
Средний дневной спрос за 30 дней: продажи_30д ÷ 30. Используется для расчёта дней покрытия и рекомендации. Кликните для сортировки.
Выручка 30д
Суммарная выручка за последние 30 дней. Кликните для сортировки — удобно приоритизировать отгрузку по фактическому вкладу в оборот.
₽/шт
Средняя цена продажи: выручка 30д ÷ кол-во 30д. Помогает выделить дорогие товары — они важны даже при низком ABC по штукам. Кликните для сортировки.
Рек.
Суммарная рекомендация к отгрузке по всем кластерам. Формула: продажи_28д × буфер_ABC − сток − в_пути. Буфер: A+Зелёный = 1.5×, A/B+Жёлтый = 1.25×, остальные = 1×. Кликните для сортировки.
Статус
🔴 Дефицит — остаток < 7 дней. 🟡 Риск — < 14 дней или есть рекомендация. 🟢 Норма — всё хорошо. 🔵 Избыток — остаток > 90 дней и рекомендация = 0. Кликните для сортировки.
Грузим
Пометить товар как активный для отгрузки. Неактивные товары можно скрыть фильтром «Только Грузим». Кликните для сортировки.
