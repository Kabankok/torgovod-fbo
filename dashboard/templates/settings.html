Где взять ключи?

  1. Зайдите в личный кабинет продавца Ozon (seller.ozon.ru).
  2. Настройки → Seller API → «Сгенерировать ключ».
  3. Выберите роль Admin read only (или Admin) и скопируйте Client-Id и Api-Key.

Инструмент только читает данные и помогает планировать поставки. Ключи никуда не отправляются, кроме официального API Ozon.
